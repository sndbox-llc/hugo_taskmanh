1. タイトルは自由入力
2. 担当者は初期設定「２　スタッフ登録」で登録したスタッフから選択
3. タスクステータスは「ステータス編集」で登録したステータスから選択→「④　ステータス編集」参照
4. カテゴリは「カテゴリ編集」で登録したカテゴリから選択→「⑤　カテゴリ編集」参照
5. 案件番号は、案件登録されている案件番号から選択（選択したい案件番号を入力することもできます）→案件登録は「案件管理」→「案件登録」参照
